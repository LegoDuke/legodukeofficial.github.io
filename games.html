// GAMES

VAPORWAVE

VAPORWAVE is a synthwave inspired 3D platforming video game, developed by myself in Unity game engine. The goal of the game is to traverse the level while collecting cards. Each card gives you more abilities to perform, which help you win the game.

Clumsy Kaiju

Klumsy Kaiju is a 3D real-time strategy game that was made by me and my team in one week for the Godot wild game jam. I was the co-lead programmer for the game, and it was developed in Godot game engine using GD Script programming language.

Court of Laughs

Court of Laughs is a 2D deck building roguelike game that was made by me and my team in one week for a game jam. I was the co-lead programmer for the game, and it was developed in Unity game engine using C# programming language.

Killer Kitty

Killer Kitty is a campy 2D top-down shooter game I made with a team of two other people. The game was developed for Godot Wild game jam, using the Godot game engine. I was the lead programmer for the game and it was developed in GD Script.
